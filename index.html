Niche Market Research Tool

read-only personal use Etsy Open API v3
Personal developer project

Market research for one prospective Etsy seller.

This is a small command-line tool I built for myself. Once a day it queries public Etsy listings by keyword, aggregates listing counts, price ranges, and shop statistics, and writes a local report that helps me decide which handmade and digital products to create and sell on Etsy. It never modifies anything, collects no user data, and is used only by me.

How it works

STEP 1 — QUERY

Search public listings

A fixed list of ~18 seed keywords is searched once daily against public, active listings.

STEP 2 — AGGREGATE

Compute category stats

Listing counts, price distributions, and public shop statistics are combined into a per-keyword opportunity score.

STEP 3 — REPORT

Write a local file

Results are saved as a CSV and a short Markdown summary on my own machine. Nothing is published or resold.

API usage

  • GET/v3/application/listings/active Keyword search of public active listings (findAllListingsActive).
  • GET/v3/application/shops/{shop_id} Public shop statistics for context (getShop).
  • GET/v3/application/listings/batch Public listing details in batches (getListingsByListingIds).

Scope & compliance

Access

API key only — no OAuth, no member data

Writes

None. Read-only endpoints exclusively

Volume
